pyFormex is a python implementation of Formex algebra.
pyFormex lets you generate large geometrical models of 3D structures by a sequence of mathematical transformations.
It is especially suited for the automated design of spatial frame structures. But as the examples show,
it can also be used for other tasks, like finite element preprocessing, or just for creating some nice pictures.

Prerequisites

For running pyFormex, you need to have the following packages installed:

  • Python : version 2.2 or higher
  • Numerical Python (aka. NumPy) : version 0.9.4 or higher
  • PyOpenGL : version 2.0 or higher
  • PyQt : version 0.9 or higher

If all you want to do is use the pyFormex mathematical transformations, the first two packages should be sufficient.
If you want to use the graphical interface and create images, you will also need the other two.

While pyFormex is developed on Linux platforms, it should be possible to get it running on other platforms too, provided you have installed the above packages.
