Download pyFormex

You can get pyFormex from one of the following sources:
  • Official releases are available from the developer pages. For most users this is the prefered way get pyFormex.
  • Occasionally pre-releases with new functionality are available from our local FTP server.
  • The latest source is always available from the SVN repository.
  • If you can not install all the prerequisites but would still like to try out the program, you can run pyFormex directly from the BuMPix Linux Live CD.
