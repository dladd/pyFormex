Screenshots

  • Stent unroll procedure: This screenshot shows four stages in the evaluation of the geometry of a stent device.
  • Transparency: The transparent mode can be useful for many applications, such as the one shown in this screenshot. By using the transparent mode, it is possible to visualize the complete stent structure, which is implanted in a diseased artery.
  • Wire stent: This screenshot shows the main stages during the generation of the wire stent geometry.
  • Surface mesh manipulations: The top viewports show a stent device (acquired from CT scan images) in smooth rendering. At the bottom, some cross section are shown which have been created with pyFormex.
  • Fluid simulations: pyFormex can also be used to create grids for Computational Fluid Dynamics (CFD). Again, many parameters can easily be modified (anlge of intersection, number of boundary layers,...).
