\chapter{pyFormex --- reference manual}
\label{cha:reference}

This is the reference manual for \pyformex.


The definitions in the modules \module{coords}, \module{formex}, \module{script} and (for GUI applications) \module{draw} are available to your \pyformex scripts without explicitly importing them. Also available is the complete \module{numpy} namespace.

All the other modules need to be accessed using the normal Python import statements.
