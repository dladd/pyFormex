34. nurbs — Python equivalents of the functions in lib.nurbs¶

The functions in this module should be exact emulations of the external functions in the compiled library.

Classes defined in module nurbs

Functions defined in module nurbs

nurbs.binomial(n, k)¶

Compute the binomial coefficient Cn,k.

This computes the binomial coefficient Cn,k = fac(n) / fac(k) / fac(n-k).

>>> print [ binomial(4,i) for i in range(5) ]
[1, 4, 6, 4, 1]
